Future<Uint8List> readWrite({required int indexGroup, indexOffset, readLength, required Uint8List writeData, Duration? timeout}) async {
  final payload = buildReadWritePayload(indexGroup: ..., indexOffset: ..., readLength: ..., writeData: ...);
  final response = await _command(AdsCommandId.readWrite, payload, timeout); // throws on AMS errorCode
  final decoded = decodeReadWriteResponse(response);
  _throwOnResult(decoded.result); // throws on outer ADS result
  return decoded.data;
}
```
`_command` = single AMS-throw site; `_throwOnResult` = single outer-ADS-result throw site. Both throw AdsException.fromCode. Constants: `AdsIndexGroup.sumUpRead/sumUpWrite/sumUpReadWrite`, `AdsCommandId.readWrite`.
