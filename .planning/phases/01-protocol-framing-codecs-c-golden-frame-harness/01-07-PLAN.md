| T-1-02 | Tampering | endianness regressions | mitigate | CI grep gate fails the build if any multi-byte accessor in `lib/src/protocol/` omits `Endian.little`. |
| T-1-EXP | Information Disclosure | over-broad public export | mitigate | Barrel uses intentional `export ... show` / curated exports; internal-only types (raw buffers, helpers) are not exported. |
</threat_model>
