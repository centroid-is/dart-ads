<!-- Reassembly contract. A complete on-wire frame = [6B AMS/TCP][32B AMS][payload]. -->
<!-- AMS/TCP length field (u32 @ offset 2, little-endian) = 32 + payload = everything after the 6-byte wrapper. -->
<!-- So total frame bytes = 6 + length. -->
FrameAssembler:
  FrameAssembler({int maxFrameBytes = 4 * 1024 * 1024})   // 4 MiB default, configurable
  List<Uint8List> add(Uint8List chunk)  // append, then emit every complete frame now available
  // if buffered >= 6: read length; if length > maxFrameBytes -> throw MalformedFrameException (before allocating)
  //   if buffered >= 6 + length: slice one frame, emit, advance; else keep remainder and return
Consumes: lib/src/protocol/ams_tcp_header.dart + exceptions.dart (plan 01-04); test/golden/*.hex (plan 01-02); test/support/hex.dart (plan 01-01).
No dart:async, no dart:io anywhere in lib/src/protocol/.
