Write 0x03          | req: group u32, offset u32, length u32 + data[length] | resp: result u32 = 4B

WriteControl 0x05   | req: adsState u16, devState u16, length u32 + data[length] | resp: result u32 = 4B
ReadWrite 0x09      | req: group u32, offset u32, readLen u32, writeLen u32 + writeData[writeLen] | resp: result u32 + readLength u32 + data[readLength]
