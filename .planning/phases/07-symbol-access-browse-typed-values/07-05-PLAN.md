## Trust Boundaries

| Boundary | Description |

| device → client | Handles + blob bytes returned; stale handle reuse could hit the wrong symbol |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-7-01 | DoS | handle lifecycle | mitigate | readByName/writeByName release in try/finally; AdsHandle.close() idempotent; no leak on op failure |
| T-7-05 | Tampering | AdsHandle reuse after staleness | mitigate | 0x710/0x711 marks the handle invalid; a later op throws StateError (no silent reuse against a reallocated symbol) |
| T-7-02b | DoS | browseSymbols nSymSize | mitigate | Sanity-cap nSymSize before allocating the 0xF00B read buffer (device-controlled length) |
| T-7-SC | Tampering | package installs | accept | No package installs this phase |
</threat_model>
