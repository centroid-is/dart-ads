<objective>
Implement the subscribe verb (CLI-04): open a device-notification Stream for a
symbol (by name) or (group,offset,len), print each sample as a timestamped line
until the operator interrupts with Ctrl-C, then tear the notification handle down
cleanly on SIGINT so the PLC never leaks a handle (the CONTEXT's headline
correctness property).

Purpose: subscribe is the live-value verb and the single most leak-prone path —
a dropped SIGINT teardown orphans a PLC notification handle. Output: one command
file + one integration test asserting zero active handles after teardown.
</objective>
